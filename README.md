# hugo-helper
hugo helper

## Install

```bash
npm install -g u10/hugo-helper
hh
```

## Build
```bash
git clone https://github.com/u10/hugo-helper.git

cd hugo-helper

npm install

npm run build
```

## Development
```bash
git clone https://github.com/u10/hugo-helper.git

cd hugo-helper

npm install

npm run coffee

# start server
./bin/hh

# serve with hot reload at localhost:8080
npm run dev
```
